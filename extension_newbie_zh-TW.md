# 🧩 如何安裝 Vault 助手擴充功能 (Extension)

如果你從未手動安裝過瀏覽器套件，請參考以下步驟。只需一分鐘即可完成！

## 1. 準備套件資料夾
首先，請確保你已經下載了專案代碼（請參考 [主指南](README_zh-TW.md#getting-started)）。
在專案目錄中找到名為 `grok-extension` 的資料夾。

---

## 2. 將套件載入瀏覽器

### Chrome, Edge 或 Brave 用戶
1. 打開瀏覽器，進入 `chrome://extensions/` (或是 `edge://extensions/`)。
2. **關鍵步驟：** 開啟右上角的 **「開發者模式 (Developer mode)」** 開關。
3. 點擊 **「載入解壓縮 (Load unpacked)」** 按鈕。
4. 選擇你剛下載的專案資料夾中的 `grok-extension` 資料夾。
5. **成功！** 你現在應該能在工具列看到「Vault 助手」的圖示了。

### Firefox 用戶
1. 打開 Firefox，進入 `about:debugging#/runtime/this-firefox`。
2. 點擊 **「載入暫時性附加元件... (Load Temporary Add-on...)」**。
3. 導覽至 `grok-extension` 資料夾，隨便選一個裡面的檔案（例如 `manifest.json`）。
4. **注意：** Firefox 的暫時性套件在重啟瀏覽器後會消失。若要永久安裝，則需經過 Mozilla 的正式簽署（進階操作）。

---

## 3. 開始使用
1. 前往 `https://grok.com/imagine`。
2. 畫面右下角會出現一個可愛的小面板 **「Vault 助手」**。
3. 如果沒看到，請重新整理頁面，或點擊工具列上的套件圖示。
