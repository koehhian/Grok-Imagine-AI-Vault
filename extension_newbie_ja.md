# 🧩 Vault 助手（拡張機能）のインストール方法

ブラウザ拡張機能（Extension）を手動でインストールしたことがない方でも、以下の手順に従えば1分で完了します！

## 1. 拡張機能フォルダの準備
まず、プロジェクトのコードをダウンロードしていることを確認してください（[メインガイド](README_ja.md#getting-started)を参照）。
プロジェクトフォルダの中に `grok-extension` という名前のフォルダがあるはずです。

---

## 2. ブラウザへの読み込み

### Chrome, Edge, Brave をお使いの場合
1. ブラウザを開き、`chrome://extensions/`（または `edge://extensions/`）にアクセスします。
2. **重要：** 右上にある **「デベロッパー モード」** をオンにします。
3. **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4. ダウンロードしたプロジェクト内の `grok-extension` フォルダを選択します。
5. **成功！** ツールバーに「Vault 助手」のアイコンが表示されます。

### Firefox をお使いの場合
1. Firefox を開き、`about:debugging#/runtime/this-firefox` にアクセスします。
2. **「一時的なアドオンを読み込む...」** をクリックします。
3. `grok-extension` フォルダ内のファイル（`manifest.json` など）をどれか一つ選択します。
4. **注意：** Firefox の一時的な拡張機能は、ブラウザを再起動すると消えてしまいます。常用する場合は、Mozilla による署名が必要です（上級者向け）。

---

## 3. 使い方
1. `https://grok.com/imagine` にアクセスします。
2. 画面右下に **「Vault 助手」** の小さなパネルが表示されます。
3. 表示されない場合は、ページを更新するか、ツールバーの拡張機能アイコンをクリックしてください。
